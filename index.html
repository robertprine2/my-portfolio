Progress in UCF Full Stack Web Developer Boot Camp

Module 1: Mastering the Browser

Length: 6 weeks

Week 1: In the first week, we were introduced to GIT/GitHub, taken through HTML, and learned the basics of CSS. The homework was to create a basic portfolio website that matched pictures given to us (changing personal information to show who we were).

Week 2: In the second week, we focused on CSS in depth, including bootstrap and how to make websites responsive. We covered a lot of different ways to manipulate a webpage and were encouraged to dig deeper on our own. The homework was to make our basic portfolio website responsive and to create a unique portfolio website, using bootstrap or coding on our own, that we could use to highlight our strengths and future projects to prospective employers.

Week 3: In the third week, we learned the basics of Javascript. We covered different variables including arrays and objects. We then used those variables to make functions and methods to produce basic games. The homework was to make a themed hangman game using what we have learned so far in Javascript.
